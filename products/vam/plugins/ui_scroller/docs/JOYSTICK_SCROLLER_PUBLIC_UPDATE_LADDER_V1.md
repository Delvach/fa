# Joystick Scroller Public Update Ladder V1

## Purpose

This is the sandbagged public update ladder for Joystick Scroller.

The goal is not to pretend these are giant feature leaps.

The goal is:

1. keep the FrameAngel brand visible
2. ship a few clean public updates
3. let the scroller serve as a low-risk exposure lane while the new player
   product family is being stood up

## Update Ladder

## Rule

Do not turn this lane into a deep feature lab while the player family is being
architected.

Use it as:

1. a shipping proof lane
2. a brand touchpoint
3. a clean low-risk update cadence
